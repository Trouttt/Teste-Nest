### User Update Use Case

## Caso de sucesso

1. Sistema valida os dados
2. Sistema faz uma requisição para a URL da api
3. Sistema efetua o update

# Exceção - Não conseguiu achar o usuário para efetuar a atualização - Válidado

1. Sistema retorna um statusCode e uma mensagem de erro avisando que não encontrou um usuário para efetuar a atualização

# Exceção - Não conseguiu efetuar a atualização - Válidado

1. Sistema retorna um statusCode e uma mensagem de erro avisando que deu erro ao efetuar a atualização
